
# Bash File and Directory Game

A simple interactive terminal game built with Bash scripting for my operating systems / shell programming practice. It's designed to help test and practice basic Linux command-line navigation, file paths, and directory management.

---

## What the Game Does

* **Choose Your Mode:** You can choose whether you want to test yourself on **files**, **directories**, or **both**.
* **Select Number of Rounds:** Before starting, the game asks you how many rounds you want to play.
* **Directory Path Checking:** In each round, the script checks your main directory to see if the file or directory path you entered is correct.
* **Track Scores:** It shows you your progress after each round and displays a summary score at the end when you finish all rounds.

---

## Requirements

* Any Linux/Unix environment (Ubuntu VM, WSL, macOS terminal, etc.)
* Standard `bash` shell

---
